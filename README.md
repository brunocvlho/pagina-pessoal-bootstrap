# Página Pessoal - Atividade Prática 2

## 📋 Descrição

Página Pessoal desenvolvida em HTML, CSS e JavaScript, conforme requisitos da Atividade Prática 2. O projeto demonstra o uso de dois frameworks CSS distintos: **Bootstrap** e **Tailwind CSS**(https://github.com/brunocvlho/pagina-pessoal-tailwind).

## 📁 Estrutura do Projeto

```
├── bootstrap/           # Versão com Bootstrap
│   ├── index.html      # Página principal
│   ├── style.css       # Estilos personalizados
│   └── script.js       # JavaScript
    └── README.md       # Este arquivo
```

## ✨ Funcionalidades

- ✅ **Layout Responsivo**: Adapta-se a diferentes tamanhos de tela
- ✅ **Tema Claro/Escuro**: Toggle entre temas com persistência em localStorage
- ✅ **Frameworks CSS**: Bootstrap
- ✅ **Seções Completas**:
  - Apresentação pessoal
  - Sobre mim
  - Formação acadêmica
  - Projetos
  - Contato

## 🚀 Como Usar

### Versão Bootstrap

1. Abra `bootstrap/index.html` em um navegador
2. Clique no botão 🌙/☀️ para alternar o tema

## 🛠️ Tecnologias Utilizadas

### Bootstrap

- Bootstrap 5.3.0 (CDN)
- HTML5
- CSS3 com variáveis CSS
- JavaScript Vanilla

## 🎨 Tema Claro e Escuro

Ambas as versões possuem:

- **Tema Claro**: Paleta clara com fundo branco
- **Tema Escuro**: Paleta escura com fundo cinzento/preto
- **Persistência**: O tema selecionado é salvo no `localStorage` e carregado ao retornar

### Paleta de Cores - Bootstrap

**Tema Claro:**

- Background: Branco (#ffffff)
- Texto: Chumbo (#212529)
- Navbar: Cinza claro (#f8f9fa)

**Tema Escuro:**

- Background: Preto (#1a1a1a)
- Texto: Cinza claro (#e0e0e0)
- Navbar: Preto (#212121)

## 📱 Responsividade

Ambas as versões são totalmente responsivas.
